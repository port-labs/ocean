# Claude Managed Agents

An integration used to import [Claude Managed Agents](https://platform.claude.com/docs/en/managed-agents/overview) resources into Port, keep them fresh via webhooks, and trigger or create agents from Port workflows.

#### Install & use the integration - [Integration documentation](https://docs.port.io/build-your-software-catalog/sync-data-to-catalog/ai-agents/claude-managed-agents/)

#### Develop & improve the integration - [Ocean integration development documentation](https://ocean.getport.io/develop-an-integration/)
