## 0.1.0-beta (2026-01-30)

### Features

- Initial Harbor integration with support for projects, users, repositories, and artifacts
- Pagination support for all Harbor API endpoints
- Webhook support for real-time artifact events
- Basic auth support for Harbor robot accounts and local users
