# Cursor Cloud Agents

An integration used to import [Cursor Cloud Agents](https://cursor.com/docs/cloud-agents) resources into Port, keep them fresh via webhooks, and create or trigger agents from Port workflows.

#### Install & use the integration - [Integration documentation](https://docs.port.io/build-your-software-catalog/sync-data-to-catalog/ai-agents/cursor-cloud-agents/)

#### Develop & improve the integration - [Ocean integration development documentation](https://ocean.port.io/developing-an-integration/)
